# Package Installed

- mongoose
- express 
- cors 
- dotenv 
- bcrypt 
- cookie-parser 
- jsonwebtoken

## To run 
- create a .env file
PORT=5000
MONGODB_URI=your_mongo_connection_string
ACCESS_TOKEN = your_access_token
ACCESS_TOKEN_EXPIRY = your_access_token_expiry
REFRESH_TOKEN = your_refresh_token
REFRESH_TOKEN_EXPIRY = your_refresh_token_expiry

npm install
